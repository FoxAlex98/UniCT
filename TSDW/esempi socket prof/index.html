Istruzioni per la compilazione

Per compilare il sorgente file.c, invocare: make file
Il file Makefile definisce alcune dipendenze tenute in conto nella compilazione.

Semplice client/server finger

fingerc.c (usa mkaddr()); si provi: ./fingerc directory.mit.edu smith > mit.log
e si noti, in mit.log, come i byte vengono ricevuti a blocchi)
fingers.c (Finger server: si provi, con il finger client fingerc, a riprodurre l'effetto osservato con mit.edu)

Semplice client/server nel dominio Unix

unclnt.c unsrvr.c (con gestione terminazione)

IP addresses/names - Resolver/DNS

mkaddr.c (utility mkaddr() per la costruzione di un indirizzo di socket nel dominio internet dati nome host e port)
addrcnv.c (usa inet_aton())
printaddr.c (routine per stampare una struct hostent del resolver)
byaddr.c byname.c (semplici invocazioni della routine gethostby...() del resolver)
gethost.c (invoca le gethostby...() del resolver)

Socket non bloccanti

nblksock.c

Semplici client-server nel dominio Internet

insrvr.c
inskelsrvr.c, insrvr.c semplificato con meno controlli di errore, adatto per svolgere esercizi
doserv.c inclnt.c
getmsg.c getmsgTout.c (ricezione bloccante, con tout (select()), non bloccante)

Server parallelo

inparsrvr.c

Applicazione client/server con socket

mboxclnt.c mboxsrvr.c

Send e Receive: connectionless e non

sndrcvClessClnt.c sndrcvClessSrvr.c
sndrcvConnClnt.c
sndrcvConnClntStream.c sndrcvConnSrvrStream.c
